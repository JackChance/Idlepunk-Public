# Idlepunk-Public

## Found a Bug or Have a Suggestion?

You don't need to know anything about coding to report a problem or share an idea — all you need is a free GitHub account.

**Here's how to open an issue:**

1. Make sure you're logged into [GitHub](https://github.com). If you don't have an account, you can create one for free.
2. Click the **Issues** tab near the top of this page (it's right next to the word "Code").
3. Click the green **New issue** button.
4. Give your issue a short title that describes the problem or idea (e.g. "Game crashes when I click the upgrade button" or "Idea: add a dark mode").
5. In the big text box below the title, describe the issue in as much detail as you can. The more context you provide, the easier it is to help. For bugs, it helps to include:
   - What you were doing when the problem happened
   - What you expected to happen
   - What actually happened instead
   - Any error messages you saw
6. When you're ready, click **Submit new issue**.

That's it! Someone from the project will follow up in the issue thread. You'll get a notification by email if you have them enabled.
